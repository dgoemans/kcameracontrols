## Typography

### Font Specifications

- **Application Font**: System default (typically Noto Sans)
- **Effect Title**: Bold, 11pt
- **Labels**: Regular, 10pt
- **Button Text**: Regular, 10pt
- **Tooltips**: Regular, 9pt

## Spacing and Layout

### Main Window
- Window margin: 15px all sides
- Element spacing: 10px vertical
- Effect row height: ~50px
- Button size: 30x30px (icon buttons)
- Corner radius: 3-4px

### Effect Rows
- Internal padding: 10px horizontal, 5px vertical
- Title on left
- Buttons on right (5px spacing between them)
- Border: 1px solid #bdc3c7
- Shadow: subtle (1px offset)

## Responsive Behavior

### Minimum Window Size
- Width: 600px
- Height: 500px

### Maximum Window Size
- Width: unlimited (reasonable max ~1200px)
- Height: unlimited (scrollable content)

### Resizing
- Effect panel scrolls vertically if too many effects
- Minimum comfortable height: 400px
- Window can be maximized

## Accessibility Features

### Keyboard Navigation
- Tab: Move between elements
- Enter: Activate button/open dropdown
- Space: Toggle/activate
- Arrow keys: Navigate dropdowns and sliders
- Escape: Close dialogs

### Screen Reader Support
- All buttons have accessible names
- Sliders announce current value
- Dropdowns announce selection
- Status changes announced

### High Contrast Mode
- Respects system theme
- Higher contrast borders in HC mode
- Clear focus indicators

## Animation (Subtle)

- Button hover: 100ms fade
- Menu open: 150ms slide
- Dialog open: 200ms fade
- No jarring animations
- Can be disabled in system settings

This mockup represents the fully implemented UI with all planned features.
